## 2. 제품 CTA만 제거하면 해결되는 페이지 수

**46건 전부**입니다.

기존 감사 데이터(`mechanism_overreach_without_cta` 필드)를 재확인한 결과, 46개 D등급 페이지 중 CTA 블록 **바깥**(본문 교육 콘텐츠 자체)에서 기전→효능 과잉 연결이 발견된 사례는 0건이었습니다. 즉, D3(기전 과잉 연결)와 D4(미확정 제품 확정 노출)는 전부 `.cta-banner` 블록 **내부**에 갇혀 있으며, 이 블록 하나를 제거하면 D1·D2·D3·D4가 동시에 해소되는 구조입니다.

> ⚠️ 주의: 이 판단은 기존 감사에서 자동 추출한 `mechanism_overreach_without_cta` 플래그와 CTA 블록 자체의 재확인에 근거한 것이며, 46개 아티클 본문 전체를 문장 단위로 재검토한 결과는 아닙니다. 최종 확정 전 최소 표본 재확인을 권장합니다(§11 감사 한계 참고 — 본 하위 감사 자체의 한계).

## 3. 의료 본문 재검수가 필요한 페이지 수

**0건 (현재 확보된 데이터 기준)**. CTA 블록을 제외한 본문 교육 콘텐츠 자체에서 근거 대비 과잉 주장이 자동 탐지되지 않았습니다. 다만 이는 §2와 동일한 한계를 가지므로, "본문 자체는 문제 없다"를 확정 결론이 아닌 **잠정 결론**으로 취급해야 합니다.

## 4. 저자명만 수정하면 되는 페이지 수

**단독으로 "저자명만" 수정하면 끝나는 페이지는 0건**입니다 — 46건 전부 D1(제품 CTA)이 우선 해결되어야 하므로, 저자명 수정이 단독 완결 조치가 되는 경우는 없습니다.

다만 D5 내부에서 **저자명 표기 불일치**(`Dr. Claire Ham (Jeehyun Ham, M.D.)` → `Jeehyun Ham, M.D.` 표준)가 실제로 필요한 페이지는 **22건, 전부 영어판**입니다. 한국어 24건은 이전 세션에서 이미 "Jeehyun Ham, M.D."로 통일 완료되어 있음을 재확인했습니다(`grep`으로 `Dr.H` 잔존 여부 재검증, 0건).

나머지 46건 전체에 공통되는 D5 하위 문제는 "의료 검수자·검토일 필드 부재"이며, 이는 저자명과 무관하게 구형 아티클 템플릿 구조 자체의 한계입니다(신형 템플릿은 `✅ 의학 검토:` 줄을 별도로 갖추고 있으나, 이 46건은 그 이전 템플릿으로 작성됨).

- D2(질환명 연결) 분류는 제목/슬러그 기반 키워드 매칭으로 수행했으며, 본문 전체를 정독하여 맥락을 재확인한 것은 표본적으로만 수행했습니다.
- "의료 본문 재검수 필요 0건"이라는 §3의 결론은 기존 감사의 자동 플래그(`mechanism_overreach_without_cta`)에 의존한 것으로, 46건 전체를 문장 단위로 재검토한 결과가 아닙니다.
- 참고문헌(§ evidence-box) 자체는 46건 전부 0건으로 확인했으나(원 감사 CSV `reference_count` 필드), 이는 "참고문헌이 없다"는 사실 확인이며 별도의 근거 적정성 판단은 하지 않았습니다.
- 9-2의 "중복 게시 추정"은 제목 유사성에 근거한 정성적 판단이며, 실제 본문 내용 중복 여부(표절/재사용 수준)까지 대조하지는 않았습니다.
- 거버넌스 문서 06, 08, 11~15, `W02`~`W05`는 이번 하위 작업에서 다시 열람하지 않았습니다(상위 감사에서도 미열람이었던 항목과 동일).
